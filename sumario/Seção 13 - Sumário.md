## Visão Geral da Seção
A Seção 13 - Machine Learning reúne 18 aulas que constroem os fundamentos e práticas aplicadas deste domínio na AWS, com foco direto na preparação para a certificação AWS Data Engineer Associate.

## Conceitos Principais
- Serviços centrais abordados na seção: SAGEMAKER.
- Princípios de arquitetura de dados na AWS com foco em escalabilidade, segurança e custo.
- Decisões práticas para o exame AWS Data Engineer Associate (DEA-C01), incluindo trade-offs entre serviços.
- Boas práticas de operação, troubleshooting e integração entre componentes de dados.
