Portanto, a ideia é que você possa comprar

E o AWS, é claro, o Data Exchange.
Assim, depois de assinar um produto de
dados,

diretamente no Amazon S3 e, em seguida,
pode

esses

Assim, por exemplo, digamos que queremos
obter todos

que,
por sua vez, despejará os dados no Amazon
S3 e, em seguida, poderemos usar, por
exemplo,
algo como o SageMaker para executar o
aprendizado
de máquina com base no conjunto de dados
do Foursquare, tudo isso a partir do AWS.
E há uma pequena função de pesquisa
no catálogo de navegação para procurar,
por exemplo, mais de 4.000 produtos
de dados disponíveis no Data Exchange.
Outros recursos do Data Exchange são o
Data Exchange for Redshift, no qual você
assina dados de terceiros do Data Exchange
novamente e, em seguida, carrega-os
diretamente de
um data warehouse do Amazon Redshift e
pode consultar os dados diretamente do
Redshift.
E isso significa que todos vocês também
podem fazer o caminho inverso.
Se você tiver dados no Amazon Redshift,
poderá licenciar
os dados por meio do AWS Data Exchange.
